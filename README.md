
# 🇯🇵 GPick 日本代購連線管理工具

專為日本現場連線設計，支援 AI 智慧辨識與手機快速拍照上架。

## 📲 如何將最新程式碼更新到您的 GitHub？

為了讓您的手機版呈現最新功能，請按照以下步驟手動更新：

1.  **開啟 GitHub**：進入您的 `zola87/gpick-app` 儲存庫。
2.  **更新檔案**：
    *   找到 `metadata.json` -> 點擊編輯 (鉛筆圖示) -> 貼上新程式碼 -> 點擊下方的 **Commit changes**。
    *   進入 `成分` 資料夾 -> 點擊 `LiveSession.tsx` -> 點擊編輯 -> 貼上新程式碼 -> 點擊下方的 **Commit changes**。
    *   (如果有變動) 更新 `App.tsx`。
3.  **等待自動部署**：
    *   如果您是使用 Vercel 或 GitHub Pages，通常在提交 (Commit) 後 1 分鐘內，手機開啟網頁就會自動變成最新版本。
